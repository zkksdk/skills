## 技术栈与架构
- 49实例运行 AgentHub：NestJS REST API(:3000) + 独立 Socket.IO 服务(:3001) + Vue3 前端
- Socket.IO 独立服务器方案：避免 NestJS Express catch-all 中间件拦截 WebSocket 请求
- nginx 反向代理：/api → :3000，/ws → :3001（HTTPS upstream）
- 前端连接策略：相对路径 + window.location.origin，移除所有硬编码域名
- JWT 验证 secret 必须与 NestJS 一致（`agenthub-secret-2026`），否则 socket 认证失败立即断开

## 已解决的关键问题
